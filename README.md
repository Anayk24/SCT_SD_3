# 🧩 Java Sudoku Solver
## Project Description

The Java Sudoku Solver is a console-based program that automatically solves a Sudoku puzzle using an efficient algorithm. The program takes a partially filled 9×9 Sudoku grid (with zeros representing empty cells) and fills in the missing numbers so that the final solution follows all Sudoku rules.

This project demonstrates the use of backtracking, 2D arrays, recursion, and validity checks in Java.

## Features

✔ Automatically solves a valid Sudoku puzzle
✔ Uses backtracking algorithm for systematic exploration
✔ Checks row, column, and sub-grid validity
✔ Displays the solved Sudoku board in the console
✔ Simple to customize and extend

## Algorithm Used

The solver uses Backtracking, which:

Tries numbers 1–9 in each empty cell

Checks whether each number is valid (not in the same row, column, or 3×3 box)

Recursively solves deeper and backtracks if needed

This ensures the solution is found correctly and fully.
